很长一段时间没有用git了，前几天使用git上传文件，又重新查找了一下git的使用方式，整理在这里

  1. git init —在本地建立git仓库（会在文件夹下建立.git隐藏文件）
  2. git add . —将当前文件夹下的所有文件加载到本地暂存区
  3. git commit -m “init commit” —提交到本地历史区
  4. git remote add origin https://github.com/YuanYunjing/shequkaifa0.0.git —建立一个origin的远程仓库关联得到XX url
  5. git push -u origin master —向git提交（在使用中报错![rejected]，后来发现是有一个超过100M的文件，删除文件后，可以成功执行了）

    （6. find ./ -size +100M —如何找到大于100M的文件）
  1. git pull –rebase origin master —若新建远程仓库的时候自动创建了README文件会报错，需先将github上的该文件拉下来，防止冲突
  2. git push -u origin master
